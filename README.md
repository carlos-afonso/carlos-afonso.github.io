## Open Source Projects

### Visualizing the 2019 Measles Outbreak

Helping understand the 2019 measles outbreak by transforming raw data into visualizations (examples below).

See all visualizations in the project webpage: [https://carlos-afonso.github.io/measles](https://carlos-afonso.github.io/measles){:target="&#95;blank" rel="noopener"}

All code available in the GitHub repository: [https://github.com/carlos-afonso/measles](https://github.com/carlos-afonso/measles)

__PyData NYC 2019 tutorial:__ Presented part of this project as a tutorial entitled _"Visualizing the 2019 Measles Outbreak in NYC (with Python)"_ at the PyData NYC 2019 conference [https://pydata.org/nyc2019](https://pydata.org/nyc2019){:target="&#95;blank" rel="noopener"}. Links to all tutorial materials are available in the GitHub repository [README.md page](https://github.com/carlos-afonso/measles/blob/master/README.md).
